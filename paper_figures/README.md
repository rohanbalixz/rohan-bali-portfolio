# Paper Figures

Add figures and visualizations from your papers here.

Reference them in your content as:

```markdown
![Figure](/paper_figures/your-figure.png)
```

**Recommendations:**
- High resolution (300 DPI for prints)
- Clear, readable labels
- Format: PNG for diagrams, JPG for photos
- File size: < 1MB
- Descriptive filenames

**Naming convention:**
- `paper-name-figure-1.png`
- `neurips2024-architecture.png`
- `urban-growth-results.png`
