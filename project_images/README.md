# Project Images

Add your project screenshots here. Reference them in `data/projects.json` as:

```json
"image": "/project_images/your-image.jpg"
```

Recommended image sizes:
- Width: 800-1200px
- Height: 400-600px
- Format: JPG or PNG
- File size: < 500KB (optimize for web)

Naming convention:
- Use lowercase
- Use hyphens instead of spaces
- Be descriptive: `election-forecasting-dashboard.jpg`
